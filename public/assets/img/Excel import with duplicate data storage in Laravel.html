excel import and duplicate data sore in another database in laravel

To import Excel data and store duplicate data in another database in Laravel, you can follow these steps:

  1. First, you need to create a new Laravel project and set up the database connections for both the primary and secondary databases in the config/database.php file.

  2. Next, you can use a package like Maatwebsite/Laravel-Excel to handle the Excel import process. You can install this package using composer by running the following command:

    bash
    composer require maatwebsite/excel

  3. After installing the package, you can create a new controller and add a method to handle the Excel import. In this method, you can use the Excel facade provided by the package to read the Excel data and store it in the primary database.

  4. To check for duplicate data, you can retrieve the data from the secondary database and compare it with the imported data. If there are any duplicates, you can store them in the secondary database.

  5. To store the duplicate data in the secondary database, you can create a new connection to the secondary database using the DB::connection() method provided by Laravel. Then, you can use the insert() method to insert the duplicate data into the secondary database.

Here's an example code snippet that shows how you can import Excel data and store duplicate data in another database in Laravel:

php
use Illuminate\Http\Request;
use Illuminate\Support\Facades\DB;
use Maatwebsite\Excel\Facades\Excel;
use App\Http\Controllers\Controller;

class ExcelImportController extends Controller
{
    public function import(Request $request)
    {
        $data = Excel::toArray([], $request->file('excel_file'));

        $primaryData = [];
        $secondaryData = [];

        // Loop through the imported data
        foreach ($data as $row) {
            // Check if the data already exists in the primary database
            if (!DB::table('primary_table')->where('column', $row['column'])->exists()) {
                // If not, add it to the primary data array
                $primaryData[] = $row;
            } else {
                // If it already exists, add it to the secondary data array
                $secondaryData[] = $row;
            }
        }

        // Insert the primary data into the primary database
        DB::table('primary_table')->insert($primaryData);

        // Insert the secondary data into the secondary database
        $secondaryConnection = DB::connection('secondary_database');
        $secondaryConnection->table('secondary_table')->insert($secondaryData);
    }
}

Note that this is just an example and you may need to modify it to fit your specific requirements.
